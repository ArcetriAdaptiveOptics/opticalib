opticalib documentation
=======================

**opticalib** is a Python package for adaptive optics laboratory instrumentation
control and deformable mirror (DM) calibration.  It provides:

* Easy connections to interferometers (4D PhaseCam, AccuFiz) and deformable
  mirrors (Alpao, SPLATT, AdOptica).
* Routines for DM Influence Function (IFF) acquisition and processing.
* DM flattening, modal decomposition, and wavefront analysis utilities.
* Simulated devices for offline development and testing.

.. note::
   This project is maintained by the `Adaptive Optics Group`_ at
   INAF – Osservatorio Astrofisico di Arcetri.

.. _Adaptive Optics Group: https://ao.arcetri.inaf.it/
